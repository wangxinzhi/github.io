Git笔记

Git命令(LOCAL)

  1. 初始化仓库代码
    • git init
  2. 添加文件到暂存区
    • git add
  3. 提交文件
    • git commit -m
  4. 查看工作区状态
    • git status
  5. 比较不同
    • git diff
  6. 显示提交日志
    • git log <–preety=oneline>
  7. 回退到上个版本
    • git reset –hard <HEAD^>
  8. 显示你的每次命令
    • git reflog
  9. 让文件回到最近一次add或commit时的状态
    • git checkout –file
  10. 把暂存区的修改撤销，重新放回工作区
    • git reset HEAD filename
  11. 创建并切换分支
    • git checkout -b branchname
  12. 查看分支
    • git branch
  13. 创建分支
    • git branch
  14. 切换分支
    • git checkout
  15. 合并分支到到当前分支
    • git merge
  16. 删除分支
    • git branch -d
  17. 查看提交记录
    • git log –graph
  18. git remote add origin
  19. git branch –set-upstream local/branchname origin/branchname

Git命令(REMOTE)

  1. 本地仓库关联远程仓库
    • git remote add origin git@github.com:github账号名/项目名
  2. git push -u origin master
  3. git clone git@github.com:项目地址
  4. git checkout -b dev origin/dev
  5. git pull

实际开发中操作Git

  1. 克隆远程仓库代码到本地
    • git clone git@github.com:项目地址
  2. 创建并切换到本地dev分支
    • git checkout -b branchname
  3. 提交本地代码至远程仓库
    1. 仓储
    2. 拉取远程分支代码
    3. 弹出仓储代码（有冲突解决冲突，无冲突万事欧克）
    4. add本地代码至stage
    5. commit stage中的代码
    6. pull
